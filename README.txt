APIs Used:

- IATA Codes Database (http://iatacodes.org/) - to get airport codes based on city
- QPX Express API (https://developers.google.com/qpx-express/) - to get flight suggestions
- Google Maps Geocoding API (https://developers.google.com/maps/documentation/geocoding/start) - to get the GPS coordinates of the destination, for weather
- Dark Sky API (https://darksky.net/dev/) - to get the weather forecast for the destination on the travel date

To run:
- Open the project in Netbeans
- Run the project (F6 or right-click + Run)
- Fill in the form fields to search for flights and display the weather forecast.
